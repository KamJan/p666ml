Depths of Listening

A reflective tool to map and articulate how deeply you experience music

This tool turns your music-level table into something you can read, edit, compare, and now also discuss with a visible chat assistant inside the page.

Purpose of this tool: to help map how deeply music is experienced, give language to different listening states, and make that inner experience easier to explain to yourself and to others.

Hybrid mode: Satan works now with local interpretive logic and is also ready for an external API endpoint later.
Level
depth of listening
Intensity

Context

How to use it

1. Edit any cell. 2. Use the chat to describe what the music did to you. 3. The Satan (local) suggests vocabulary, body cues, after-effects, and likely levels. 4. Below you can find your Satan helper (powered by OpenAI). 5. Export CSV or print when done.

About Satan chatbot

You can use the Satan chatbot as a focused “experience translator”: write what happened to you with the music, and let it suggest language, body cues, and likely levels that you can then copy into the table.
